# DHU Web Promotion Site

このリポジトリは、デジタルハリウッド大学 Web 専攻のプロモーションサイトのフロントエンドコードです。  
GitHub Pages 上で公開しており、以下の URL からご覧いただけます。  
https://himeragiinmilkxx.github.io/dhuweb/

元になっている公式サイトはこちらです。  
https://www.dhw.ac.jp/faculty/web/

## 概要

本サイトは、4 年次に受講した授業の最終課題として制作したものです。  
これまで複数の Web 系授業で制作物を作ってきましたが、その中でも特に人に見せたいと思える、納得度の高いアウトプットになっています。

デジタルハリウッド大学の Web 専攻の魅力を、ビジュアルとインタラクションを通して伝えることを目的としたプロモーションサイトです。

## 使用技術

- フロントエンド：React.js, typescript, tailwindcss, shadcn/ui, Three.js  
- デザイン：Figma
